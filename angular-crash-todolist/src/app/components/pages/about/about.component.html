About

This is the TodoList app v1.1.0. It is based on a Angular crash course

The crash course is available on youtube.com/Fdf5aTYRW0E

The original project can be found on github.com/bradtraversy/angular-crash-todolist



Changes on the original project

  • Created own api using Django Rest Framework
  • Added the functionality to edit the Todo text
  • Minor layout changes, such as the checkbox which wasn't checked when an item was already completed
